GDI San Jose class materials

This site hosts a variety of CC-licensed web development curriculum which was developed for Girl Develop It San Jose. This curriculum is designed to be taught, but it may be useful to students as well. This table lists all the curriculum:

Topic Prereqs Duration Other links
test & test None 2hrs Meetup description | Lesson plan | Follow Up Email

Interested in teaching?

  • Take a look at GDI's class materials and see if you're interested in teaching any of these.
  • You may also develop a course of your own!
  • Consult our teaching guide.
  • Try doing the exercises yourself and make sure you understand what issues students might run into while doing them.
  • After you teach it, report issues and open Pull Requests on the Github project.
  • Let us know about what you taught where.

Interested in contributing? Fork our curricula on Github!
